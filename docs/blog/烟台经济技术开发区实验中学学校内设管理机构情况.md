|机构名称|政教处|办公电话|0535-3450862|
|机构职能|1.负责全校德育工作，指导班主任工作。负责选聘班主任和班级辅导员,做好青年班主任的培养工作。  <br><br>2.指导班主任制订班级工作计划。<br><br>3.培训学生干部。<br><br>4.指导年级主任、班主任召开班级家长会。<br><br>5.定期召开班主任工作会议,组织学习、总结、交流班主任工作经验。<br><br>6.协助年级主任、班主任处理年级、班级中的重大问题。<br><br>7.根据学校奖惩制度,做好学生的奖惩工作,树立正气,打击歪风邪气，维护校风校纪。<br><br>8.协助总务处做好校园周边环境综合治理工作，负责落实教室卫生的督查工作。|   |   |
|机构名称|教务处|办公电话|0535-3450836|
|机构职能|1.教务处是学校教学工作的指挥中枢，负责全校的教学组织和领导工作，并负责处理一切与教学有关的事务性工作。<br><br>2.根据教育部或上级教育行政部门颁发的教学计划、课程标准和教学意见，制定学校每学期的教学工作计划。<br><br>3.负责管理各年级教学工作，指导各年级的教学活动，落实教学常规。<br><br>4.负责全校的教学常规工作的检查落实工作。组织学生评教工作。<br><br>5.负责日常的教务工作，包括文印工作、学生的学籍管理工作、考务工作、课表的安排、中招咨询和录取工作、学生的学籍管理工作、教师档案管理工作。<br><br>6.有计划地组织教师参加校内外各种形式的业务进修，不断提高教师的业务水平。|   |   |
|机构名称|教科处|办公电话|0535-3450839|
|机构职能|1. 教科处依据教改计划，落实于各教研组。<br><br>2.各教研组长依本组的学科特征制定教研计划，每月安排至少两次教研时间，要求全体教师必须参加，各抒己见。<br><br>3.各教研组必须按时按量的完成教研计划，并把好的经验总结，与其他教研组进行交流、沟通，完善、健全各组的活动。<br><br>4.教研活动有教研组长负责，记录活动内容、参加人数、考勤，期末上交教科处统一存档。<br><br>5.教科处与学校行政领导每月对各教研组活动进行检查，并认真听取各教研组长的工作情况汇报，掌握教科研活动开展的情况。|   |   |
|机构名称|安全办|办公电话|0535-3450847|
|机构职能|1.及时传达贯彻学校安全领导小组的工作要求，负责学生的安全教育和宣传。注重学生入学教育，加强遵纪守法、安全防范、饮食卫生防疫和校规校纪教育。<br><br>2.建立健全安全管理规章制度，制定安全事故预防措施和事故抢险预案。检查监督本部门所辖的安全工作落实情况。<br><br>3.负责与各职能岗位签订安全责任书，将安全责任落实到有关人员，并考核其执行情况。<br><br>4.定期开展安全检查。排查校园危房，排查隐患，落实整改措施，保证师生员工人身和财产安全。并对本岗无力解决的问题如实上报学校安全领导小组研究处理。<br><br>5.做好学生集体活动的安全防范工作。实行学生集体活动报告制度，切实落实学生集体活动安全责任制。<br><br>6.遇突发事件发生时，要及时赶到现场，组织人力积极稳妥地迅速作好师生的撤离、善后处理工作。<br><br>7.按照上级要求，做好学生保险工作。|   |   |
|机构名称|总务处|办公电话|0535-3450820|
